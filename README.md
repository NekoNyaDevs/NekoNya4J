## 🐱 NekoNya4J

### 📝 About

NekoNya4J is a Java wrapper for the [NekoNya API](https://nekonya.classy.works).

### 📚 Usage

#### 📦 Maven

```xml
<dependency>
    <groupId>com.classycode</groupId>
    <artifactId>nekonya4j</artifactId>
    <version>1.0.0</version>
</dependency>
```

#### 📦 Gradle

```groovy
implementation 'org.nekonya:nekonya4j:1.0.0'
```
